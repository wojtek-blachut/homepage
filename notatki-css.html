Notatki CSS

Co to jest CSS?

Są to kaskadowe arkusze stylów, odpowiadają za warstwe prezentacji. Nadają kolory czcionką, tłom itp. Umieszczamy je w osobnym pliku do którego dołączamy link w headzie, ów plik zwykle ma rozszerzenie CSS. Typpowy CSS wygląda następująco:

selektor{
właściwośc:wasrtość;
}

Wszystkie prawa zastrzezone.
